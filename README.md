## 🛠 Tech Stack

- **Frontend:** React.js, Context API, Tailwind CSS / CSS3
- **Backend:** Node.js, Express.js
- **Database:** Static JSON Data (Custom API)
- **Notification:** React-Toastify
- **Icons & Images:** Lucide-react / Font-Awesome & Unsplash/Pexels
